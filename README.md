This folder contains code for the paper "Energy-Aware Holistic Optimization in UAV-Assisted Fog Computing: Attitude, Trajectory, and Task Assignment" and is associated with the GitHub repository https://github.com/Shuaijun-LIU/HATTO-UFog.


Online demo: https://shuaijun-liu.github.io/UAV-Assisted-Fog-Computing-Simulation-Demo.

The demo is a lightweight, browser-based (Three.js) interactive visualization created for the paper to help reviewers quickly understand the simulation environment in a simplified setting.
