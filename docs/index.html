Overview

We study a UAV-enabled fog computing system where a single UAV acts as both a mobile relay and a fog node. The framework jointly optimizes attitude control, trajectory planning, resource allocation, and task assignment, ensuring decisions in one module improve outcomes in the others.

Attitude Control

FEAR-PID improves stability for takeoff, cruising, and landing under dynamic conditions.

Trajectory Planning

ACS-DS adds decoupling and safety values to accelerate convergence and avoid local optima.

Task Assignment

PSO-based allocation balances computation between IoT devices, UAV, and cloud.

Joint Optimization

Cross-layer decisions reduce latency and energy consumption across the full mission.

Holistic Framework

The system integrates control, networking, and computation modules in a tightly coupled loop. The UAV attitude influences communication quality, which feeds trajectory updates and task offloading decisions.

Key Contributions

  • Unified optimization across attitude, trajectory, resource allocation, and task assignment.
  • FEAR-PID for adaptive, stable UAV attitude management.
  • ACS-DS trajectory planning with decoupling and safety-aware search.
  • PSO-based offloading and resource allocation under realistic constraints.
