Overview of the UAV-assisted fog computing environment with joint attitude, trajectory, and task assignment optimization.

Abstract

Unmanned Aerial Vehicles (UAVs) have significantly enhanced fog computing by acting as both flexible computation platforms and communication mobile relays. We consider four interdependent modules: attitude control, trajectory planning, resource allocation, and task assignment, and propose a holistic framework that jointly optimizes total latency and energy consumption for UAV-assisted fog computing in a three-dimensional spatial domain with varying terrain elevations and dynamic task generations. We establish a fuzzy-enhanced adaptive reinforcement PID control model for attitude management, propose an ACS-based trajectory algorithm with decoupling and safety values, and design a PSO-based algorithm for task execution placement. Extensive simulations show significant latency and energy improvements over mainstream approaches.

Method Overview

Attitude Control

FEAR-PID maintains stable UAV orientation under dynamic flight conditions to protect link quality.

Trajectory Planning

ACS-DS integrates decoupling and safety values to avoid deadlocks and accelerate convergence.

Task Assignment

PSO-based allocation decides where tasks execute across devices, UAV, and cloud.
